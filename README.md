# 🧠 AI-Based Diabetes Prediction System

A machine learning-powered web application that predicts the risk of diabetes based on user health parameters.

---

## 🌐 Features

- Predicts diabetes risk using ML model  
- Probability-based risk output  
- Risk categorization (Low / Medium / High)  
- User-friendly interface using Streamlit  
- Data visualization (charts & comparisons)  
- Feature importance analysis  

---

## 📌 Problem

Diabetes is a growing health issue, and early detection is crucial. Many people are unaware of their risk due to lack of accessible tools.

---

## 💡 Solution

This system allows users to input basic health parameters and instantly receive a prediction along with insights and recommendations.

---

## ⚙️ Tech Stack

- Python  
- Streamlit  
- Scikit-learn  
- Pandas  
- Matplotlib & Seaborn  

---

## 📊 Input Features

- Age  
- BMI  
- Physical Activity  
- High Blood Pressure  
- High Cholesterol  
- Cholesterol Check  

---

## 🎯 Output

- Diabetes Risk Prediction  
- Probability Score  
- Risk Level (Low / Medium / High)  
- Personalized Suggestions  

---

## 🚀 How to Run

bash
pip install -r requirements.txt
streamlit run app.py
